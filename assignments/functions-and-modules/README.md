# 📘 Assignment: Functions and Modules

## 🎯 Objective

Learn how to write reusable functions, use parameters and return values, and organize code with a main program structure.

## 📝 Tasks

### 🛠️ Create Reusable Functions

#### Description

Write three functions in `starter-code.py` that each perform a small task and return a result.

#### Requirements

Completed program should:

- Define a function `calculate_rectangle_area(width, height)` that returns the area of a rectangle.
- Define a function `format_greeting(name)` that returns a greeting string such as `Hello, Alice!`.
- Define a function `is_prime(number)` that returns `True` when the number is prime and `False` otherwise.
- Include example calls to each function in the `main()` section.

### 🛠️ Use a Main Program Structure

#### Description

Organize the program so it can be imported without running the example code immediately.

#### Requirements

Completed program should:

- Define a `main()` function that calls the helper functions and prints their results.
- Use `if __name__ == "__main__":` to run `main()` only when the file is executed directly.
- Keep function definitions separate from the program entry point.
